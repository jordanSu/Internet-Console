# Internet-vim
## Description
This is OS lab1

## Requirments
* One client, one server
* when client connect to server, type the command below can do:
  - (C) create a new file
  - (E) edit a file (check the file is exist first)
  - (R) remove a file
  - (L) list all the file
  - (D) download the file

## Extensions function
1. Files Encryption/Decryption
2. batch downloading
3. Menu
